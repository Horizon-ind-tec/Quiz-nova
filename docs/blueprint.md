# **App Name**: QuizNova

## Core Features:

- Custom Quiz Generation: Generate quizzes based on user-selected class, subject, educational board, and difficulty level using AI.
- Multiple-Choice Quiz: Allows users to answer multiple-choice questions.
- Answer Review: Shows correct answers and explanations after quiz completion.
- Performance Tracking: Tracks the user's quiz scores and progress over time and adjusts the generation tool accordingly.

## Style Guidelines:

- Primary color: Vibrant blue (#3498DB), reflecting intelligence and focus.
- Background color: Soft white (#F9F9F9) providing a clean, distraction-free environment.
- Accent color: Orange (#F39C12) to highlight important elements like buttons and progress indicators.
- Body and headline font: 'Inter', a grotesque-style sans-serif, provides a modern and readable appearance.
- Use clean, minimalist icons in the vibrant blue color to represent different subjects and features.
- Clean and intuitive layout with a focus on readability. Use white space effectively to avoid clutter.
- Subtle transitions and animations to provide feedback and guide the user through the app.
